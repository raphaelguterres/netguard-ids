🔐 Segurança — Admin Token

O token de admin fica em .netguard_token. Rotacione se suspeitar de vazamento ou periodicamente como prática de segurança. O token antigo é invalidado imediatamente e o novo é exibido uma única vez.

Ação destrutiva — requer re-login após rotação.
🔒 Segurança — 2FA (TOTP)

Ative 2FA opcional para o login admin. Secret em .netguard_totp. Compatível com Google Authenticator, 1Password, Authy.
